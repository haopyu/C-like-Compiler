# C-Like-Compiler

## Description
A C-like Compiler
* Using Lex and Yacc to do lexical analysis and parsing
* Construct an abstract syntax tree
* Construct a symbol table
* Generate IR
## Environment
* Ubuntu Linux
* Flex(Lex) 2.6.0
* Byacc(Yacc) 1.9

## Build
>$>make  
>$>.\QAQ \<filename\>
